# JavaScript_Projets

In this repository, as a student I will be learning and implementing the ability to modify web pages using JavaScript, jQuery, Bootstrap, React.js and more.


Projects;
-Pizza Menu
-Tic Tac Toe Game
-Calculator

Pizza Menu:
This project is a menu used to order pizza with a various selection of toppings. Then taking the input information and calulates a price for said items.

Tic Tac Toe:
The classic game of tic-tac-toe. It randomly rolls dice to select which of the two players goes first and then users are off playing the game. Users decide which token each player wants to use and the winner is congratulated with music and lighting.

Calculator:
A basic four function calculator. Users can add, subtract, multiply and divide on a sleek web version calculator.
